# Atividade Extra 2

Esse repositório contém arquivos da atividade.

## Como rodar

Clone o repositório:  

`git clone https://github.com/aksaraymalaklisi/atividade_extra_backend_2.git`

Com Docker instalado, acesse o a pasta do repositório e inicialize os contêineres com Docker Compose:  

`docker compose up -d`  

A página estará rodando em `localhost:8080`.
